multiplayer platform shooter game made using socket.io

## Usage
Clone repo and
```javascript
npm install
```
to install dependencies and after
```javascript
node server
```
to run the server

## Controls
```
w-s-a-d - moving 
aiming - mouse
shooting - lmb
t - stats
